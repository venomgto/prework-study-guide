HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The !DOCTYPE declaration represents the document type, and helps browsers to display web pages correctly. It must only appear once, at the top of the page (before any HTML tags). The !DOCTYPE> declaration is not case sensitive.
  • HTML headings are defined with the h1 to h6 tags. h1 defines the most important heading. h6 defines the least important heading.
  • HTML paragraphs are defined with the p tag.
  • HTML links are defined with the a tag.
  • HTML images are defined with the img tag. The source file (src), alternative text (alt), width, and height are provided as attributes

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much we want around the content inside an element.
  • A color can be specified by using a predfined color name.
  • The CSS border properties allow you to specify the style, width, and color of an element's border.
  • We can divide css selectors into five categories:
    • Simple selectors (select elements based on name, id, class)
    • Combinater elements (selects elements based on a specific relationship between them)
    • Pseudo-class selectors (select elements based on certain state)
    • Pseudo-elements selectors (select and style a part of an element)
    • attribute selectors (select elements based on an attribute or attribute value)

Git

  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A
  • git commit -m "text based on what you're working on"
  • git pull origin main
  • git push origin "branch you are working on name"
  • git checkout main
  • git pull origin main

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • A string is for storing and manipulating text
  • Variables can be declared in 4 ways(automatically, var, let, const)
  • Loops are handy, if you want to run the same code over and over again, each time with a different value
